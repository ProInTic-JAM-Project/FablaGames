# FablaGames

Catálogo de videojuegos y creadores locales de Aragón. Combina un seed local con datos e imágenes de RAWG. Cada tarjeta abre un drawer con detalles y carrusel de capturas. Incluye modo oscuro.

## Requisitos
- Node.js
- API key de RAWG en `.env.local`:
  - `RAWG_API_KEY=tu_clave`

## Desarrollo
```bash
npm install
npm run dev
```

Abre http://localhost:3000
